Detailed Methodology

1. Data Cleaning & Imputation

Missing values in the dataset were handled systematically. Numeric columns were filled using the median value, while categorical columns were filled using the mode.

2. Feature Engineering

Categorical features (Time of Day, Weather, Traffic, etc.) were transformed into numeric format using Label Encoding to ensure compatibility with regression algorithms.

3. Model Training & Selection

We evaluated three distinct models using an 80/20 train-test split:

  • Decision Tree Regressor: Achieved an 80.85% R² score but showed signs of overfitting.
  • SVR (Support Vector Regression): Performed poorly on this dataset with only a 44.22% R² score.
  • Random Forest Regressor: The final selected model, utilizing 200 estimators to achieve a superior 93.17% R² score and a low MAE of 5.36.
